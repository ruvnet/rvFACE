rvFACE

Face recognition SDK in Rust + WebAssembly — Burn inference (CPU + WebGPU), detector → landmark → embedder pipeline, running entirely in your browser. No telemetry, local-only processing.

The interactive WASM demo is being built and will deploy here shortly. Meanwhile: clone the repo and run ./web/build-wasm.sh for the local demo, or cargo run -p rvface-cli for the CLI.
